## Installation & Setup
Clone the repository and install dependencies:
```bash
git clone https://github.com/T0ASTY/Budsy.git
cd Budsy
npm install
```

Create a `.env` file and add your Discord bot token:
```
TOKEN=your_discord_bot_token
```

Run the bot:
```bash
node index.js
```
